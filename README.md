# 📊 Automação de Conciliação Financeira (Case Real)

Este projeto automatiza o cruzamento de dados entre planilhas de comissões e a base geral de vendas, tarefa que anteriormente era realizada manualmente.

## 🛠️ Tecnologias Utilizadas
- **Python 3.x**
- **Pandas**: Para manipulação e limpeza de dados.
- **XlsxWriter**: Para geração de relatórios Excel com formatação profissional.

## 🚀 Funcionalidades
- **Normalização:** Limpeza automática de códigos de reserva e nomes.
- **Mapeamento Dinâmico:** Identificação de colunas mesmo com variações de nomes (ex: Fornecedor/Vendor).
- **Relatório Automático:** Gera um arquivo final pronto com abas separadas por colaborador.

## 📈 Impacto
- Redução do tempo de processamento de horas para segundos.
- Eliminação de erros manuais de cruzamento (VLOOKUP).
